# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of LogLens
- Interactive log viewer dashboard
- FastAPI integration
- Flask integration
- Plotly-based chart visualizations
- HTML table generation
- Log parser with pattern detection
- Log analyzer with statistics
- Search functionality
- Filter by log level, logger, and custom queries
- API endpoints for log access and analysis
- CLI tool for quick log summary
- Comprehensive documentation
- MIT License

### Features
- Real-time log visualization
- Error frequency tracking
- Top loggers analysis
- Log level distribution
- Beautiful responsive UI
- Mobile-friendly design
- Support for common log formats
- Custom regex pattern support
